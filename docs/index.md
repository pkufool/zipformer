# Zipformer

A faster and better encoder for ASR (Automatic Speech Recognition).

## Overview

Zipformer is a novel neural network architecture designed for speech recognition tasks. It provides improved accuracy and inference speed compared to previous encoder architectures.

## Quick Start

```bash
pip install zipformer
```

```python
import zipformer
```
